# Poll: Atlas Bootstrapping Edits - July 1, 2024

The Governance Facilitators have placed a Governance Poll into the voting system using their powers defined in [MIP113 12.1.1](https://mips.makerdao.com/mips/details/MIP113#12-1-1). This Governance [Poll](https://manual.makerdao.com/governance/governance-cycle/weekly-governance-cycle#weekly-governance-cycle-definitions-mip16c1) will be active for three days beginning on Monday, July 1 at 16:00 UTC.

**This is a binary vote.**

- **You may vote for a single option.**
- **You should vote for the option which you prefer.**
- **If you would accept either option, you should vote 'Abstain'.**

## Review

The community can vote in this poll to express support or opposition to the changes in the following Github Pull Request:

- [Amendment Pull Request](https://github.com/makerdao/mips/pull/1114).

Please review the discussion [thread](https://forum.makerdao.com/t/atlas-bootstrapping-edit-proposal/24557/1) to help inform your position before voting.

## Outcomes

**If the votes for the 'Yes' option exceed the votes for the 'No' option then the following actions will be taken:**

- The proposed changes will be merged into the Atlas.

**If the votes for the 'No' option equal or exceed the votes for the 'Yes' option then no further action will be taken at this time.**
